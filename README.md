# Data Pipeline Portfolio

A modern, interactive portfolio website built with Astro that visualizes my professional journey as a data engineering pipeline.

## 🚀 Core Concept

This portfolio reimagines the traditional portfolio structure as a data pipeline, where visitors navigate through stages of a data processing journey while learning about my skills and experience.

## 📊 Pipeline Structure

The portfolio follows a data pipeline architecture with these stages:

1. **Ingestion** – Introduction (who I am, profile, tagline, goals)
2. **Validation** – Skills, tools, certifications, and strengths
3. **Transformation** – Key projects, code samples, before/after use cases
4. **Storage** – Repositories, resume downloads, tech diagrams or schemas
5. **Orchestration** – Side projects, workflows, CI/CD setups, team roles
6. **Monitoring** – GitHub stats, metrics, testimonials, activity feeds
7. **Delivery** – Contact form, call-to-action, external links

## ✨ Features

- **Interactive Pipeline Navigation**: Vertical sidebar showing pipeline progress
- **Data Flow Visualizations**: Animated elements representing data moving through the pipeline
- **Stage-Specific Theming**: Each pipeline stage has its own color scheme and visual identity
- **Responsive Design**: Works on all device sizes
- **Dark Mode Support**: Full light/dark theme implementation
- **Multilingual Support**: Available in English and Spanish

## 🛠️ Technology Stack

- **[Astro](https://astro.build/)**: Core framework for building the static site
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework for styling
- **[astro-i18next](https://github.com/yassinedoghri/astro-i18next)**: Internationalization support

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |

## 🎨 Design Elements

- **Pipeline Flow Indicators**: Visual elements showing data moving through stages
- **Progress Tracking**: Interactive sidebar showing current position in the pipeline
- **Stage Transitions**: Smooth animations between pipeline stages
- **Data Nodes**: Visual representations of skills, projects, and experience as data points
- **Terminal & Dashboard Aesthetics**: Technical visuals appropriate for a data engineering portfolio

## 📱 Responsive Behavior

- **Desktop**: Full pipeline visualization with interactive elements
- **Tablet**: Adapted layout with preserved pipeline flow
- **Mobile**: Simplified vertical pipeline with optimized content display
